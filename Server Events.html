    • Start
    This event being called when a 'Start' operation ended.
    int code, string message
    0, "Started."
    1, "Already started!"
    2, "Cannot create socket!"
    3, "Cannot create listen thread!"
    4, "Cannot bind socket!"
    5, "Cannot start listen thread!"
    6, "An error occured!"
    Stop
    This event being called when a 'Stop' operation ended.
    int code, string message
    0, "Stopped."
    1, "Already stopped!"
    2, "An error occured!"
    OptionChange
    This event being called when a 'OptionChange' operation ended.
    int code, string message, ServerOption option
    0, "X changed.", option
    1, "X must be higher than Y!", option
    2, "X must be lower than Z!", option
    99, "Server must be stopped to change any option!", option
    Login
    This event being called when a client attempted to establish connection.
    uint clientId, EndPoint endPoint, byte[] bytes
    If this call returned 0 (zero), that means login accepted. If this call returned a byte that is not 0 (zero), that means login refused with this return value.
    Connect
    This event being called when a connection setup ended.
    uint clientId, int code, string message
    clientId, 0, "Client connected."
    clientId, 1, "Client peer setup failed!"
    Disconnect
    This event being called when a client dropped from the server.
    uint clientId, int code, string message, byte reasonCode
    0, "Client disconnected.", reasonCode
    1, "An error occured! (client-side)", 0
    2, "An error occured! (server-side)", 0
    3, "Network locked! (client-side)", 0
    4, "Network locked! (server-side)", 0
    5, "Connection timed out! (client-side)", 0
    6, "Connection timed out! (server-side)", 0
    7, "Server dropped the connection!", 0
    8, "Too high ping! (server-side)", 0
    9, "Client peer setup failed!", 0
    10, "Server stopped!", 0
    Receive
    This event being called when server successfully received some data from a connected client.
    uint clientId, byte[] data, int dataLength, PacketType type
    Send
    This event being called when a 'Send' operation ended with an error.
    int code, string message, uint clientId
    0, "Server must be running to send data!", 0
    1, "Cannot perform a successful send operation because data is null or empty (zero length)!", 0
    2, "Client not found!", clientId
    PingUpdate
    This event being called when server successfully completed a ping-pong latency calculation for a connected client.
    uint clientId, uint ping
